Module configuration options

Configure Options

  • name: AHRS_FINV_TYPE value: AHRS_PRIMARY|AHRS_SECONDARY
    Description: set if the AHRS is the primary source (default) or one of the secondary sources
  • name: USE_MAGNETOMETER value: TRUE
    Description: set to FALSE to disable magnetometer
  • name: AHRS_ALIGNER_LED value: 1
    Description: LED number to indicate AHRS alignment, none to disable (default is board dependent)

Define Options

  • name: SEND_INVARIANT_FILTER value: None
    Description: send INV_FILTER message for debugging
  • name: AHRS_FINV_IMU_ID value: ABI_BROADCAST
    Description: ABI sender id of IMU to use
  • name: AHRS_FINV_MAG_ID value: ABI_BROADCAST
    Description: ABI sender id of magnetometer to use
