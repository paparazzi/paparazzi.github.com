struct mavlink_message

Mavlink v1.0 message structure.

Mavlink Msg: STXMAV + len + { packet_seq + SYS/ID + COMPONENT/ID + MSG/ID + Payload } + ck_a + ck_b

Payload starts after 4 bytes.
