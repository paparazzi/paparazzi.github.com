gvf_ik module

Guidance algorithm based on inverse kinematics for path following using guiding vector fields

The path is defined implicitly by level sets, enabling precise transient control and feed-forward motion shaping. For more details, see https://arxiv.org/pdf/2502.17313 .

Example for airframe file

Add to your firmware section:

<module name="gvf_ik"/>

Module configuration options

Airframe file section

  • section name: IK prefix: GVF_IK_
    • name GAMMA_AMPLITUDE value: 0
      Description: Error feedforward amplitude
    • name GAMMA_OMEGA value: 0
      Description: Error feedforward angular velocity
  • section name: Ellipse prefix: GVF_IK_ELLIPSE_
    • name KE value: 1
      Description: Gain for the aggresivity of the gvf
    • name KN value: 1
      Description: Gain for the alignment of the vehicle with the gvf
    • name A value: 80
      Description: Horizontal axis length of the ellipse
    • name B value: 80
      Description: Vertical axis length of the ellipse
    • name ALPHA value: 0
      Description: Rotation of the horizontal axis
  • section name: Line prefix: GVF_IK_LINE_
    • name KE value: 1
      Description: Gain for the aggresivity of the gvf
    • name KN value: 1
      Description: Gain for the alignment of the vehicle with the gvf
    • name HEADING value: 0
      Description: Desired heading for the line (0 is North, 90 is East)
    • name D1 value: 0
      Description: Extra distance (w.r.t. the 1st point) to be travelled before turning around for the segment_loop
    • name D2 value: 0
      Description: Extra distance (w.r.t. the 2nd point) to be travelled before turning around for the segment_loop
  • section name: Sinusoidal prefix: GVF_IK_SIN_
    • name KE value: 1
      Description: Gain for the aggresivity of the gvf
    • name KN value: 1
      Description: Gain for the alignment of the vehicle with the gvf
    • name ALPHA value: 0
      Description: Desired heading for the line (0 is East, 90 is North)
    • name W value: 0
      Description: Frequency for the sinusoidal y=Asin(Wx + OFF)
    • name OFF value: 0
      Description: Offset for the sinusoidal y=Asin(Wx + OFF)
    • name A value: 0
      Description: Amplitude for the sinusoidal y=Asin(Wx + OFF)
