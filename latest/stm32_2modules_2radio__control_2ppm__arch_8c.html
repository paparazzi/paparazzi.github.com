Detailed Description

STM32 ppm decoder.

Input signal either on:

  • PA1 TIM2/CH2 (uart1 trig on Lisa/L) (Servo 6 on Lisa/M)
  • PA10 TIM1/CH3 (uart1 trig on Lisa/L) (uart1 rx on Lisa/M)
