eff_scheduling_nederdrone module

Interpolation of control effectivenss matrix

of the Nederdrone.

If instead using online adaptation is an option, be sure to not use this module at the same time!

Example for airframe file

Add to your firmware section:

<module name="eff_scheduling_nederdrone"/>

Module functions

Init Functions

These initialization functions are called once on startup.

  • ctrl_eff_scheduling_init()

Periodic Functions

These functions are called periodically at the specified frequency from the module periodic loop.

  • ctrl_eff_scheduling_periodic()
    • Frequency in Hz: 20.0
    • Autorun: LOCK
      Periodic function automatically starts after init and can't be stopped.
