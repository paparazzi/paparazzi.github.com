gvf_parametric module

Guidance algorithm for tracking 2D and 3D smooth trajectories that are defined by a parametric equation.

For more details we refer to https://wiki.paparazziuav.org/wiki/Module/guidance_vector_field . This algorithm needs the Eigen library, be sure you have it installed, e.g., git submodule update –init –recursive sw/ext/eigen

Example for airframe file

Add to your firmware section:

<module name="gvf_parametric"/>
